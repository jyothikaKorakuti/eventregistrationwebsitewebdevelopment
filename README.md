Event Registration Website

Overview:
A responsive website developed for a technology event. Users can view event information and register online.

Features:
Event Information
Registration Form
Required Field Validation
Email Validation
Success Confirmation Page
Responsive Design

Technologies Used:
HTML
CSS
JavaScript

Project Structure:
index.html
style.css
script.js
success.html

How to Run:
Open index.html in a browser or use Live Server in VS Code.
